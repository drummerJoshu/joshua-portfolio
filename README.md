# Joshua Mukhebi - Developer Portfolio

## 🚀 Quick Start

1. Open `index.html` in your browser or use Live Server extension in VS Code.
2. **Responsive** - Works on mobile, tablet, desktop.

## 📱 Features
- **Modern Design** - Dark theme, animations, hover effects
- **Fully Responsive** - Mobile-first design
- **Interactive** - Smooth scroll, mobile nav, contact form (mailto)
- **Sections**: Hero, About, Skills, Projects, Education, Contact
- **SEO Ready** - Meta tags, semantic HTML

## 🔧 Customization
1. **Profile Photo**: Replace `assets/profile-placeholder.jpg` with your photo (180x180px)
2. **Links**: Update GitHub/LinkedIn/Twitter URLs (search 'yourusername')
3. **Projects**: Add real demo links + GitHub repos
4. **CV**: Link `cv.pdf` to Download CV button
5. **Contact**: Form sends to your email

## 🌐 Deployment (Free)

### GitHub Pages:
```
git init
git add .
git commit -m "Initial portfolio"
git branch -M main
git remote add origin https://github.com/yourusername/joshua-portfolio.git
git push -u origin main
```
- Go to repo Settings > Pages > Deploy from branch `main` > Save
- Live at `https://yourusername.github.io/joshua-portfolio`

### Netlify (Drag & Drop):
1. [netlify.com](https://netlify.com)
2. Drag folder to deploy area
3. Instant live URL
